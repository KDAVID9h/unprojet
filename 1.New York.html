HTML
HTML (Hypertext Markup Language) est un langage de balisage utilisé pour créer des pages web. Il permet de structurer le contenu de la page en utilisant des balises qui définissent le type de contenu qu'elles contiennent.
CSS
CSS (Cascading Style Sheets) est utilisé pour ajouter de la présentation et du style à une page web en définissant la mise en forme, les couleurs, les polices, les marges et les espaces entre les éléments de la page.
